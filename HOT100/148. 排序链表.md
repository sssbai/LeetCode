# [148. 排序链表](https://leetcode.cn/problems/sort-list/description/?envType=study-plan-v2&envId=top-100-liked)

给你链表的头结点`head`，请将其按 **升序**  排列并返回 **排序后的链表**  。

```
<b>输入：</b>head = [4,2,1,3]
<b>输出：</b>[1,2,3,4]
```

```
<b>输入：</b>head = [-1,5,3,4,0]

```

**示例 3：** 

```
<b>输入：</b>head = []
<b>输出：</b>[]
```

<b>提示：</b>

- 链表中节点的数目在范围`[0, 5 * 10^4]`内
- `-10^5<= Node.val <= 10^5`

<b>进阶：</b>你可以在`O(nlogn)` 时间复杂度和常数级空间复杂度下，对链表进行排序吗？

## Solution

nlog(n)的时间复杂度，基本既可以判断是使用分治的思想，同时要求常数时间复杂度，不能修改，那么就是归并排序了

一种是递归排序，还有一种是迭代排序，

递归方法的空间复杂度为$O(\log n)$

迭代方法的空间复杂度为$O(1)$

## Code

### 递归方法

```c++
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode() : val(0), next(nullptr) {}
 *     ListNode(int x) : val(x), next(nullptr) {}
 *     ListNode(int x, ListNode *next) : val(x), next(next) {}
 * };
 */

class Solution {
public:
    ListNode* sortList(ListNode* head){
        // 如果链表为空或只有一个节点，直接返回，无需排序
        if (head == nullptr || head->next == nullptr) return head;

        // 使用快慢指针找到链表中点，将链表分为左右两部分
        ListNode* slow = head;
        ListNode* fast = head->next; // 快指针比慢指针快一步，确保 slow 最终停在左半部分尾部
        while (fast && fast->next) {
            slow = slow->next;
            fast = fast->next->next;
        }

        // 将链表从中间断开，变为两个子链表
        ListNode* tmp = slow;
        ListNode* head2 = slow->next; // 右半部分的起点
        tmp->next = nullptr; // 断开链表

        // 递归排序左右子链表
        ListNode* leftlist = sortList(head);
        ListNode* rightlist = sortList(head2);

        // 合并两个已排序的链表
        ListNode* dummy = new ListNode(); // 虚拟头节点，方便处理链表合并
        ListNode* cur = dummy;

        // 归并排序核心逻辑：按值大小合并两个链表
        while (leftlist && rightlist) {
            if (leftlist->val < rightlist->val) {
                cur->next = leftlist;
                leftlist = leftlist->next;
            } else {
                cur->next = rightlist;
                rightlist = rightlist->next;
            }
            cur = cur->next;
        }

        // 将剩余未合并的部分接上
        if (leftlist) cur->next = leftlist;
        else cur->next = rightlist;

        // 返回合并后的链表头节点
        return dummy->next;
    }
};

```

### 迭代方法

```c++
/**
 * Definition for singly-linked list.
 * 单链表的定义
 */
struct ListNode {
    int val;            // 节点的值
    ListNode *next;     // 指向下一个节点的指针
    ListNode() : val(0), next(nullptr) {}                        // 默认构造函数
    ListNode(int x) : val(x), next(nullptr) {}                  // 构造函数，指定值
    ListNode(int x, ListNode *next) : val(x), next(next) {}     // 构造函数，指定值和下一个节点
};

class Solution {
public:
    // 将链表从 node 开始，跳过 step 个节点，返回 step 后的那个节点，同时将前面的链表断开
    ListNode* split(ListNode* node, int step){
        if(node == nullptr) return node; // 空链表直接返回
        ListNode* back = node;
        int i = 0;
        while(i < step && node){  // 向后遍历 step 个节点
            i++;
            back = node;
            node = node->next;
        }
        back->next = nullptr;  // 将链表断开，前半部分独立
        return node;           // 返回后半部分链表的起始节点
    }

    // 合并两个有序链表，并返回合并后链表的头尾节点
    pair<ListNode*, ListNode*> merge(ListNode* node1, ListNode* node2){
        ListNode* dummy = new ListNode(); // 哨兵节点，便于操作
        ListNode* cur = dummy;

        // 按照节点值大小合并两个链表
        while(node1 && node2){
            if(node1->val < node2->val){
                cur->next = node1;
                node1 = node1->next;
            } else {
                cur->next = node2;
                node2 = node2->next;
            }
            cur = cur->next;
        }

        // 将剩下的部分拼接上
        if(node1) cur->next = node1;
        else cur->next = node2;

        // 找到合并链表的尾部
        while(cur->next){
            cur = cur->next;
        }

        // 返回合并链表的头结点和尾节点
        return {dummy->next, cur};
    }

    // 主函数：对链表进行自底向上的归并排序
    ListNode* sortList(ListNode* head) {
        int n = 0;
        ListNode* tmp = head;

        // 统计链表长度
        while(tmp){
            n++;
            tmp = tmp->next;
        }

        ListNode* dummy = new ListNode(); // 哨兵节点，简化边界操作
        dummy->next = head;

        // 自底向上的归并排序，每轮合并长度为 step 的子链表
        for(int step = 1; step < n; step *= 2){
            ListNode* end = dummy;        // 用于连接合并后的链表部分
            ListNode* cur = dummy->next;  // 当前处理的链表位置

            while(cur){
                ListNode* head1 = cur;              // 第一个子链表的头节点
                ListNode* head2 = split(head1, step); // 第二个子链表的头节点
                cur = split(head2, step);             // 下一轮处理的起点

                // 合并两个子链表
                auto [front, back] = merge(head1, head2);
                end->next = front; // 将合并后的链表连接到前面部分
                end = back;        // 更新 end 指针到新的尾部
            }
        }

        return dummy->next; // 返回排序后的链表头节点
    }
};

```
