# [76. 最小覆盖子串](https://leetcode.cn/problems/minimum-window-substring/description/?envType=study-plan-v2&envId=top-100-liked)

给你一个字符串 `s` 、一个字符串 `t` 。返回 `s` 中涵盖 `t` 所有字符的最小子串。如果 `s` 中不存在涵盖 `t` 所有字符的子串，则返回空字符串 `""` 。

**注意：** 

- 对于 `t` 中重复字符，我们寻找的子字符串中该字符数量必须不少于 `t` 中该字符数量。
- 如果 `s` 中存在这样的子串，我们保证它是唯一的答案。

**示例 1：** 

```
**输入：** s = "ADOBECODEBANC", t = "ABC"
**输出：** "BANC"
**解释：** 最小覆盖子串 "BANC" 包含来自字符串 t 的 'A'、'B' 和 'C'。
```

**示例 2：** 

```
**输入：** s = "a", t = "a"
**输出：** "a"
**解释：** 整个字符串 s 是最小覆盖子串。
```

**示例 3:** 

```
**输入:**  s = "a", t = "aa"
**输出:**  ""
**解释:**  t 中两个字符 'a' 均应包含在 s 的子串中，
因此没有符合条件的子字符串，返回空字符串。```

**提示：** 

- `^m == s.length`
- `^n == t.length`
- `1 <= m, n <= 10^5`
- `s` 和 `t` 由英文字母组成

**进阶：** 你能设计一个在 `o(m+n)` 时间内解决此问题的算法吗？
```

## Solution

变长滑动窗口

```c++
class Solution {
public:
    string minWindow(string s, string t) {
        unordered_map<char, int> cnt, cns;  // cnt：统计 t 中字符频率；cns：当前窗口中字符频率
        int typecount = 0;  // 需要满足的字符种类数

        // 初始化：统计 t 中字符频率，并记录字符种类数
        for (auto c : t) {
            if (cnt[c] == 0) typecount++;  // 每出现一个新字符，就增加种类数
            cnt[c]++;
        }

        int start = -1, end = s.size();  // start、end 表示当前最小窗口的起止位置
        int left = 0;  // 滑动窗口左指针

        // 滑动窗口右指针不断向右扩展
        for (int right = 0; right < s.size(); right++) {
            cns[s[right]]++;  // 把 s[right] 加入窗口

            // 如果该字符出现次数刚好满足 cnt 要求，则当前种类 typecount 减一
            if (cns[s[right]] == cnt[s[right]]) {
                typecount--;
            }

            // 当窗口已经包含了所有 t 中的字符（频率都满足）
            while (typecount == 0) {
                // 如果当前窗口比之前记录的最小窗口还小，则更新结果
                if (end - start > right - left) {
                    start = left;
                    end = right;
                }

                // 尝试收缩窗口左边界
                if (cns[s[left]] == cnt[s[left]]) {
                    typecount++;  // 收缩会导致某个字符不足，typecount 重新加 1
                }
                cns[s[left]]--;  // 减少左边字符出现次数
                left++;          // 左指针右移
            }
        }

        // 最终返回结果
        if (start < 0) return "";  // 没找到符合条件的窗口
        else return string(s.begin() + start, s.begin() + end + 1);  // 截取子串
    }
};

```

**优化版本**

```c++
//以下写法中有一个很重要的点就是：没有在t中的字符串永远不会大于0
class Solution {
public:
    string minWindow(string s, string t) {
        unordered_map<char, int> umap;  // 记录 t 中每个字符需要的次数
        int typecount = 0;  // 还需要匹配的字符种类数
        
        // 初始化所需字符的计数
        for (char c : t) {
            if (umap[c] == 0) typecount++;  // 新增一个字符种类
            umap[c]++;
        }

        int n = s.size();
        int bestStart = -1, bestEnd = n;  // 记录最优子串的左右边界
        int left = 0;

        for (int right = 0; right < n; right++) {
            char curChar = s[right];
            umap[curChar]--;  // 扫描当前字符
            if (umap[curChar] == 0) {
                typecount--;  // 当前种类满足条件
            }

            // 当前窗口已包含 t 中所有字符
            while (typecount == 0) {
                // 尝试更新最小窗口
                if (bestEnd - bestStart > right - left) {
                    bestStart = left;
                    bestEnd = right;
                }

                char leftChar = s[left];
                if (umap[leftChar] == 0) {
                    typecount++;  // 移除后该字符不满足要求
                }
                umap[leftChar]++;  // 左边界右移
                left++;
            }
        }

        // 没找到满足条件的窗口
        if (bestStart == -1) return "";
        // 返回最小窗口子串
        return s.substr(bestStart, bestEnd - bestStart + 1);
    }
};
