# [76. 最小覆盖子串](https://leetcode.cn/problems/minimum-window-substring/description/?envType=study-plan-v2&envId=top-100-liked)

给你一个字符串 `s` 、一个字符串 `t` 。返回 `s` 中涵盖 `t` 所有字符的最小子串。如果 `s` 中不存在涵盖 `t` 所有字符的子串，则返回空字符串 `""` 。

**注意：** 

- 对于 `t` 中重复字符，我们寻找的子字符串中该字符数量必须不少于 `t` 中该字符数量。
- 如果 `s` 中存在这样的子串，我们保证它是唯一的答案。

**示例 1：** 

```
**输入：** s = "ADOBECODEBANC", t = "ABC"
**输出：** "BANC"
**解释：** 最小覆盖子串 "BANC" 包含来自字符串 t 的 'A'、'B' 和 'C'。
```

**示例 2：** 

```
**输入：** s = "a", t = "a"
**输出：** "a"
**解释：** 整个字符串 s 是最小覆盖子串。
```

**示例 3:** 

```
**输入:**  s = "a", t = "aa"
**输出:**  ""
**解释:**  t 中两个字符 'a' 均应包含在 s 的子串中，
因此没有符合条件的子字符串，返回空字符串。```

**提示：** 

- `^m == s.length`
- `^n == t.length`
- `1 <= m, n <= 10^5`
- `s` 和 `t` 由英文字母组成

**进阶：** 你能设计一个在 `o(m+n)` 时间内解决此问题的算法吗？
```

## Solution

变长滑动窗口

## Code

以下是自己写的滑动窗口的内容，但是太冗余了。

```c++
class Solution {
public:
    string minWindow(string s, string t) {
        // 用于记录目标字符串 t 中每个字符需要的个数
        unordered_map<char, int> umap;
        for (char c : t) {
            umap[c]++;
        }

        // 定义最小窗口的起始和结束位置，初始为无效值
        int start = -1, end = s.size();
        // 滑动窗口的左右指针
        int left = 0, right = 0;
        // 用于记录当前窗口中字符的出现情况
        unordered_map<char, int> tmp;
        // 当前匹配到的 t 中字符的个数（只统计在 t 中存在的字符）
        int count = 0;
        // 需要匹配的总字符个数（含重复）
        int lent = t.size();

        // 扩展右边界
        for (right = 0; right < s.size(); right++) {
            tmp[s[right]]++;  // 增加当前字符计数

            // 如果当前字符是 t 中的字符，且其在窗口中的数量不超过 t 中要求的数量
            if (umap.contains(s[right]) && tmp[s[right]] <= umap[s[right]]) {
                count++;  // 有效匹配字符数加 1
            }

            // 如果当前窗口已经覆盖了所有的 t 中字符
            while (count == lent) {
                // 更新最小窗口的位置
                if (right - left < end - start) {
                    start = left;
                    end = right;
                }

                // 准备缩小窗口，判断左边界字符是否是 t 中的字符
                if (umap.contains(s[left]) && tmp[s[left]] == umap[s[left]]) {
                    count--;  // 缩小后可能不再满足匹配条件，count 减一
                }
                tmp[s[left]]--;  // 缩小窗口，左边字符移出
                left++;          // 左指针右移
            }
        }

        // 若 start 未被更新（说明没有找到符合条件的窗口），返回空串
        if (end - start > s.size()) return "";
        // 返回最小窗口子串
        return string(s.begin() + start, s.begin() + end + 1);
    }
};

**优化版本**

```c++
class Solution {
public:
    string minWindow(string s, string t) {
        unordered_map<char, int> requiredChars;  // 记录 t 中每个字符需要的次数
        int requiredTypes = 0;  // 还需要匹配的字符种类数
        
        // 初始化所需字符的计数
        for (char c : t) {
            if (requiredChars[c] == 0) requiredTypes++;  // 新增一个字符种类
            requiredChars[c]++;
        }

        int n = s.size();
        int bestStart = -1, bestEnd = n;  // 记录最优子串的左右边界
        int left = 0;

        for (int right = 0; right < n; right++) {
            char curChar = s[right];
            requiredChars[curChar]--;  // 扫描当前字符
            if (requiredChars[curChar] == 0) {
                requiredTypes--;  // 当前种类满足条件
            }

            // 当前窗口已包含 t 中所有字符
            while (requiredTypes == 0) {
                // 尝试更新最小窗口
                if (bestEnd - bestStart > right - left) {
                    bestStart = left;
                    bestEnd = right;
                }

                char leftChar = s[left];
                if (requiredChars[leftChar] == 0) {
                    requiredTypes++;  // 移除后该字符不满足要求
                }
                requiredChars[leftChar]++;  // 左边界右移
                left++;
            }
        }

        // 没找到满足条件的窗口
        if (bestStart == -1) return "";
        // 返回最小窗口子串
        return s.substr(bestStart, bestEnd - bestStart + 1);
    }
};
